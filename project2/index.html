Day 12: Pseudo-Classes & Pseudo-Elements

Master CSS interactions and advanced styling tricks

1.Pseudo-Classes

Pseudo classes are used to define the special state of an element(hover,focus,etc),without changing the HTML structure.

a) :hover

What: Applies style when a user hovers over an element.

Why: Enhances user experience with feedback like color changes.

How:

button:hover{
background-color:blue;
color:white;
}

b) :focus

What: Applies styles when an element (like input) gains focus.

Why:Helps highlight input fields,imporving accessibility.

How:

input:focus {
border:2px solid green;
outline:none;
}

2.Pseudo-Element

Pseudo-elements allow you to style parts of an element or insert content before/after without modifying HTML.

a) ::before

What: Inserts content before the element's actual content.

Why: Add labels or design elements without HTML changes.

How:

p::before{
content: "Note:";
font-weight:bold;
}

b) :: after

What: Inserts content after the main element content.

Why: Useful for footnotes, highlights,or extra info.

How:

p:: after{
content: "(Read more)";
font-style:italic;
}

Extra:CSS Variables

Custom properties (variables) make CSS reusable,readable,and easy to manage.

Why: Maintain consistency,easy global changes,and better scalalbity

How:

:root {
--primary-color:#3498db;
--font-size:16px;
}

button {
background-color:var(--primary-color);
font-size:var(--font-size);
}
